## Overview

Advanced agentic AI companion leveraging GraphRAG with Neo4j for knowledge modeling, parallel tool execution, and dynamic LLM gateway. Designed for 24/7 autonomous operation with intelligent context management and real-time web search capabilities powered by DeepSeek R1.

Production system processing **18,000+ messages** in a knowledge graph with **sub-200ms latency** for cached queries and **99.8% uptime** over 6 months.
